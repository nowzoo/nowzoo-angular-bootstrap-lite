status: Observable<string>

An observable of a string representing the modal's status. The value can be:

  • 'uninitialized' when the modal does not exist in the DOM — i.e. before it has been shown or after it's been hidden.
  • 'show' while the modal is being animated to shown — i.e. after the show.bs.modal event.
  • 'shown' while the modal is shown — i.e. after the shown.bs.modal event.
  • 'hide' while the modal is being animated to hidden — i.e. after the hide.bs.modal event.
  • 'hidden' after the hidden.bs.modal event. Note that immediately afterward the directive disposes of the modal markup and Angular bindings, and sets the status back to 'uninitialized'.
