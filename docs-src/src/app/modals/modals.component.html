Modals

Simple Bootstrap modals with ng-template.

Quick Start

Create an ng-template with the selector nzbModal.

Put Bootstrap modal markup inside this template.

Use #modalInstance="nzbModal" in the template tag to grab a reference to the directive.

Use modalInstance.show() to open the modal.

API

show(): Promise<void>

Shows the modal, returning a promise that resolves when the modal is hidden.

hide(): Promise<void>

Hides the modal, returning a promise that resolves when the modal is completely hidden (when the Bootstrap animations are done.)

handleUpdate(): void

A wrapper around the native Bootstrap $().modal('handleUpdate') method. Use this when the height of the modal changes. This makes sure the modal is displayed in the viewport correctly (i.e. that scrollbars do not suddenly appear.);
