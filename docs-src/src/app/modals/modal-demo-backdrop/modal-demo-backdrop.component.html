Modal Backdrop

By default the backdrop is shown , and clicking on it dismisses the modal.

  • To hide the backdrop altogether, add data-backdrop="false" to the .modal element.
  • To prevent clicks on the backdrop from dismissing the modal, add data-backdrop="static" to the .modal element.
