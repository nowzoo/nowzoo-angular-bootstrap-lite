Content from Template: nzbPopoverContent

If you want to take advantage of variable interpolation, or perform complex feats of logic in your popover content section, use [nzbPopoverContent]="contentTemplate" with <ng-template #contentTemplate>...</ng-template>.

The following demo displays the number of seconds since the popover was opened, and has a handy close button that calls popoverInstance.hide().

Note that the data-title attribute will override nzbPopoverContent. Omit it when you use a template.

Also note that using nzbPopoverContent automatically sets the popover's html option to true. This is by design: Angular needs markup in order to do its stuff.
