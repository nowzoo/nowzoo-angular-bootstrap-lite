Title from Template: nzbPopoverTitle

If you want to take advantage of variable interpolation, or perform complex feats of logic in your popover title, use [nzbPopoverTitle]="titleTemplate" with <ng-template #titleTemplate>...</ng-template>.

The following demo displays the number of seconds since the popover was opened, and has a handy close button that calls popoverInstance.hide().

Note that both the title and data-title attributes will override nzbPopoverTitle. Omit them when you use a template.

Also note that using nzbPopoverTitle automatically sets the popover's html option to true. This is by design: Angular needs markup in order to do its stuff.
