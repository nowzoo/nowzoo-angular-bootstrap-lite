nzbPopoverTitle

nzbPopoverTitle: TemplateRef

Optional. Use [nzbPopoverTitle]="myTemplate" with <ng-template #myTemplate>...<ng-template #myTemplate> if you want to take advantage of variable interpolation and the directive api in the popover title.

If you don't need those things, just use the title or data-title attribute.

Using nzbPopoverTitle will automatically set the html property of the entire popover to true.

The title attribute will always override nzbPopoverTitle. Omit title if you are using a template.
