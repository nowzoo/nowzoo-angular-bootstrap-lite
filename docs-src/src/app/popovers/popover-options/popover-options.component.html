nzbPopoverOptions

nzbPopoverOptions: NzbPopoverOptions

Optional. Use [nzbPopoverOptions]="{{ '{' }}...{{ '}' }}" to set the options for an individual popover. Options you do not specify will revert to the Bootstrap popover defaults.

All the Bootstrap popover options except for title, content and selector are supported:

  • animation
  • container
  • delay
  • html
  • placement
  • template
  • trigger
  • offset
  • fallbackPlacement

Additionally, you may specify animateOnDestroy. This controls whether an open popover animates to hidden when the element that it belongs to is destroyed.
