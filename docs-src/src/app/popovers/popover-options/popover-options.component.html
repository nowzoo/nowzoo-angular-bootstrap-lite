nzbPopoverOptions: NzbPopoverOptions

Optional. Use [nzbPopoverOptions]="{{ '{' }}...{{ '}' }}" to set the options for an individual popover. Options you do not specify will revert to the Bootstrap popover defaults (or your app-wide defaults if you have set them).

Bootstrap Options

All the Bootstrap popover options except for title, content and selector are supported.

Note that you may use data- attributes for these options instead of nzbPopoverOptions.

  • animation: boolean
  • container: string|false|HTMLElement
  • delay: number|{{'{'}}show:number, hide:number}
  • html: boolean
  • placement: string|((popupEl: HTMLElement, targetEl: HTMLElement) => string)
  • template: string
  • trigger: string
  • offset: number|string
  • fallbackPlacement: string|any[]
