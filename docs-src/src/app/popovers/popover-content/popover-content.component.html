nzbPopoverContent: TemplateRef

Optional. Use [nzbPopoverContent]="myTemplate" with <ng-template #myTemplate>...<ng-template #myTemplate> if you want to take advantage of variable interpolation and the directive api in the popover content.

If you don't need those things, just use the data-content attribute.

Using nzbPopoverContent will automatically set the html property of the entire popover to true.

The data-content attribute will always override nzbPopoverContent. Make sure to omit it if you want to use a template.
